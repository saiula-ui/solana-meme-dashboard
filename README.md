# Solana Meme Dashboard + Solscan

Enhanced Solana meme coin dashboard with DexScreener data and Solscan token metadata (icons, social links).

## 🚀 Quick Start

```bash
pip install streamlit pandas requests
streamlit run app.py
```

## Features
- Real-time Solana pairs from DexScreener
- Solscan integration (icons, website, Twitter, Telegram)
- Filter by age, liquidity, search
- Revival hunting for older pairs

Made with ❤️ for the Solana meme community.
